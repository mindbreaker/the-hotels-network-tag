# The Hotels Network (THN) Tag for Google Tag Manager

This is a Google Tag Manager custom template that allows you to easily implement **The Hotels Network** loader script on your website without hardcoding HTML.

It allows you to dynamically inject the `loader.js` script by simply providing your unique Partnership ID.
